Custom shader for [Reshade](https://reshade.me/) for stereo image adjustment in video glasses such as Xreal, Viture, RayNeo, etc.

### Features
* Move screens along the X and Y axes
* Add a vignette around both screens
* Crop screen edges to avoid artifacts
* Adjust gamma, contrast, luminance, black levels, white temperature

### Usage
After configuring ReShade, add the `StereoAdjust.fx` file to the `\reshade-shaders\Shaders\` folder in your game directory.
